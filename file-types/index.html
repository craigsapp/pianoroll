File types

Here is a list of file types used in the digital processing of piano rolls.

  • CIS — 1-bit image file format using run length-encoding compression, created by Richard Stibbons.
  • SCN — Note extraction files after processing CIS files.
  • MIDI — Playback files for use with MIDI synthesizers.
